# Memory

`memory/` хранит не все идеи подряд, а только:
1. **Паттерны (`accepted-patterns.md`)**, которые:
   - были осмыслены;
   - прошли хотя бы минимальную проверку;
   - дали наблюдаемую пользу;
   - стоят использования повторно.
2. **Антипаттерны (`anti-patterns.md`)**, которые:
   - были проверены;
   - доказали свою неэффективность или вредность;
   - должны быть сохранены, чтобы не тратить время на повторные проверки тех же идей ("как делать НЕ надо").

Это не папка для сырого research.
Это "активная память" для подтверждённых полезных выводов и зафиксированных ошибок. Важно, чтобы паттерны из памяти явно синхронизировались с шаблонами создания сущностей в `agents-lab` и `skills-lab`.
