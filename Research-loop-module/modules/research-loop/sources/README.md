# Sources

В эту папку складываются записи о найденных источниках по темам исследования.

Здесь не нужно хранить длинные эссе.
Нужна короткая, структурированная и повторяемая запись, которая отвечает на вопрос:
- что это за источник;
- почему он может быть полезен;
- стоит ли тратить время на review.
