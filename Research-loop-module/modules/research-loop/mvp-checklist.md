# MVP Checklist for Research Loop

Отметь этот список после первого завершённого цикла.

## Исследование
- [ ] есть одна узкая тема исследования
- [ ] собрано минимум 3 источника
- [ ] каждому источнику присвоен Trust Tier (T1–T4)
- [ ] по каждому источнику сделан review

## Proposal + Experiment
- [ ] сформулирован хотя бы один proposal
- [ ] зафиксирован Baseline (поведение до изменения)
- [ ] хотя бы один proposal проверен экспериментом
- [ ] проверены инвариантные кейсы (нет регрессий)
- [ ] результат эксперимента зафиксирован

## Завершение (Definition of Done)
- [ ] есть финальный статус: accepted / rejected / inconclusive
- [ ] принято решение: promote / reject / defer
- [ ] если accepted → паттерн добавлен в `accepted-patterns.md` и связанный шаблон обновлён
- [ ] если rejected → причина записана в `anti-patterns.md`
- [ ] если defer → пометка оставлена в `experiments/` для повторного прогона
