# Research Topic — agent-instruction-patterns

## Why this topic

На старте системы важно понять, какие паттерны реально усиливают качество agent instructions:
- роль агента;
- ограничения;
- правила выбора инструментов;
- формат результата;
- способ проверки качества;
- поведение при неопределённости.

Если удастся собрать сильный набор паттернов, это даст прямую пользу для `agents-lab`.

## Research question

Какие повторяющиеся паттерны в качественных источниках помогают писать более сильные инструкции для агентов?

## Target module

`agents-lab`

## Success criteria

- найдено минимум 3 качественных источника;
- выделены повторяющиеся паттерны;
- сформулирован хотя бы 1 внедряемый proposal;
- proposal проверен на одном узком кейсе.
