# Agents Lab

`agents-lab` — это пространство, где разрабатываются и улучшаются агенты.

Сюда должны приходить предложения, которые касаются:
- структуры agent instructions;
- ролей и ограничений агентов;
- orchestration;
- выбора инструментов;
- поведения при неопределённости;
- форматов выхода;
- self-check и validation patterns.

## Что важно

`agents-lab` не должен напрямую заполняться сырым research.
Сюда должны приходить уже оформленные и желательно проверенные предложения из `research-loop`.

## Что можно улучшать

- шаблоны инструкций
- краткую карту репозитория для grounding
- агентные роли
- паттерны разбивки задач
- tool-routing
- handoff между агентами
- требования к формату результата
