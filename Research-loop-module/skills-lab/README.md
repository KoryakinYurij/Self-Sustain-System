# Skills Lab

`skills-lab` — это пространство разработки и улучшения скиллов.

Важно: этот модуль уже существует и развивается отдельно.
Этот README нужен как точка интеграции в составе Self-Evolving System, а не как попытка заново описать всю внутреннюю структуру `skills-lab`.

## Роль в общей системе

`skills-lab` принимает:
- предложения по улучшению структуры скиллов;
- новые шаблоны;
- выводы из исследований;
- критерии качества;
- удачные паттерны из анализа сильных примеров.

## Откуда приходят улучшения

В первую очередь:
- из `modules/research-loop/`
- из ручного анализа хороших репозиториев и чужих скиллов
- из накопленного опыта внутри системы

## Важно

Если `skills-lab` уже существует локально и содержит больше информации, этот файл не должен её затирать.
Он просто объясняет место модуля в общей архитектуре.
