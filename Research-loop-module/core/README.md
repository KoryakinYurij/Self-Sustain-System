# Core

Папка `core/` содержит документы, которые определяют:
- что это за система;
- какие у неё модули;
- как они взаимодействуют;
- какие архитектурные решения уже приняты.

`core/` не хранит сырой research и не хранит артефакты экспериментов.
Его задача — фиксировать правила устройства системы.
